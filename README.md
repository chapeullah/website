## Installation
To run this project locally:

**1. Clone the repository**
```bash
git clone https://github.com/chapeullah/website.git
```
**2. Go to the project directory**
```bash
cd website
```
**3. Install dependencies**
```bash
npm install
```
**4. Start the development server**
```bash
npm run dev
```
